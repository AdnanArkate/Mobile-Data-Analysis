# Mobile Phone Data Analysis

# Files
- app.py - dashboard
- pro.py - analysis script
- Mobile_Phone_Data.xlsx - raw data
- cleaned_mobile_phone_data.xlsx - cleaned data
- requirements.txt - packages needed
- Dashboard.png - dashboard preview
- brand_counts.png - bar chart
- ratings_pie.png - pie chart
- price_vs_battery.png - scatter plot

# How to run
pip install -r requirements.txt
python pro.py
streamlit run app.py
